    • Github

    GitHub is een bedrijf met winstoogmerk dat een cloudgebaseerde Git-repository-hostingservice aanbiedt. In wezen maakt het het voor individuen en teams een stuk eenvoudiger om Git te gebruiken voor versiebeheer en samenwerking.


    De interface van GitHub is gebruiksvriendelijk genoeg, zodat zelfs beginnende programmeurs kunnen profiteren van Git. Zonder GitHub vereist het gebruik van Git over het algemeen wat meer technische kennis en gebruik van de opdrachtregel.

    Figma

    Figma is een webgebaseerde app voor grafische bewerking en gebruikersinterfaceontwerp. Je kunt het gebruiken om allerlei soorten grafisch ontwerpwerk uit te voeren, van wireframing-websites, het ontwerpen van mobiele app-interfaces, het maken van prototypen van ontwerpen, het maken van posts op sociale media en alles daartussenin.


    Figma verschilt van andere grafische bewerkingstools. Vooral omdat het rechtstreeks in uw browser werkt. Dit betekent dat u toegang krijgt tot uw projecten en kunt beginnen met ontwerpen vanaf elke computer of elk platform zonder dat u meerdere licenties hoeft te kopen of software hoeft te installeren.

    VSC

    Visual Studio Code is een lichtgewicht maar krachtige broncode-editor die op uw desktop draait en beschikbaar is voor Windows, macOS en Linux.


    Het wordt geleverd met ingebouwde ondersteuning voor JavaScript, TypeScript en Node.js en heeft een rijk ecosysteem aan extensies voor andere talen en runtimes (zoals C++, C#, Java, Python, PHP, Go, .NET).
